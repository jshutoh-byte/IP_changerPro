# ネットワーク一発切り替えPro (Network Interface Switcher Pro)

ネットワークエンジニアや社内SEが、ルーターやスイッチの設定作業を行う際のアダプター切り替え工数を「ゼロ」に近づけるための実戦用GUIツールです。

## ■ 開発背景
インフラエンジニアの現場では、自席用セグメントと作業用セグメントを頻繁に行き来する必要があります。
「コントロールパネルからネットワーク接続を開き、プロパティからIPv4を選んで…」という、古くから続く「だるい」カチカチ操作を効率化し、エンジニアが本来の業務に集中できる環境を提供するために開発しました。

## ■ 本ツールのこだわり：あえて「有線LANのみ」
本ツールは、現場での実用性を最優先し、**無線LAN（Wi-Fi）の設定機能はあえて除外**しています。
* **誤設定の防止**: 作業中に意図せず無線側の設定を書き換えてしまう事故を防ぎます。
* **ターゲットの明確化**: 物理メディアタイプ（802.3）を自動判別し、設定対象を有線LANアダプターのみに絞り込んでリストアップします。
* **ノイズの排除**: 現場のインフラ屋が本当に必要としている「物理ポートへの接続」に特化した設計です。

## ■ 主な機能
* **インテリジェント・インターフェース抽出**: 有線LANアダプターのみを自動抽出し、インターフェース選択の手間を省きます。
* **プリセット保存機能**: 現場で頻出するセグメント（例: 172.16.0.2 等）を事前に登録し、即座に反映可能です。
* **現在のIPステータス表示**: `ipconfig`を叩かずとも、現在のアダプター情報をGUI上で即座に確認できます。
* **管理者権限の自動要求**: ネットワーク設定の変更に必要な特権昇格を、起動時に自動で行います。
* **ポータブル仕様**: インストール不要。exe化してUSBメモリ等から持ち運ぶことが可能です。

## ■ 技術仕様
* **開発言語**: Python 3.x
* **GUI構成**: Tkinter (標準ライブラリ)
* **内部ロジック**: PowerShell (Get-NetIPConfiguration) および Netsh を併用
* **ビット演算**: プレフィックス長からサブネットマスクへの変換を独自実装し、NWエンジニアに馴染みのある操作感を実現
